# 03 核心用例与状态机/时序图

状态机（PaymentStatus）
- PENDING → PAID（回调确认）
- PENDING → CANCELED（取消）
- PAID → REFUNDED（退款）

核心用例
- 创建支付：校验参数 → 生成 Payment → 调用平台适配器（扫码/收银台URL）→ 落库 → 返回支付链接/二维码。
- 处理回调：保存原文 → 验签 → 去重 → 读取 Payment → 状态迁移 → Outbox 发布领域事件。
- 发起退款：校验可退款 → 调用平台退款 → 状态迁移 → 记录流水。
- 查询状态：按商户订单号/平台交易号查询。

重试与幂等
- 入站：`Idempotency-Key` + 请求幂等表（或幂等缓存）。
- 回调：事件唯一键去重（平台交易号+事件类型+时间窗）。
- 出站：Outbox 队列 + 发布确认 + 去重键。
